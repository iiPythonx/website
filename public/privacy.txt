Class A:
    1. All of these domains are handled specifically through Cloudflare Workers or some
    other Cloudflare based product. As such, the only applicable data collection happening
    is specified directly in the Cloudflare privacy policy.

    Cloudflare privacy policy -> https://www.cloudflare.com/privacypolicy/

    2. No personal information of any kind is obtained from any of these services. There
    are no sign up forms, there are no account services, there are no access logs.

    3. No data is sold. As stated above, [I] store nothing so there's nothing to sell.

Class B:

    1. These domains are proxied through Cloudflare directly to my own infrastructure. As such,
    the Cloudflare privacy policy still applies, along with my personal data collection.

    Cloudflare privacy policy -> https://www.cloudflare.com/privacypolicy/

    2. No information of any kind is collected from these services, with the strict exception of
    cmd.iipython.dev, nd.iipython.dev, and mail.iipython.dev.

        cmd.iipython.dev: full account system with profile. you are the one providing data, and
        nothing else is being collected besides what you provide. there are no access logs, no
        account locking, no ip tracking, etc. data includes username, fastest typing speed, bio,
        password hash, etc. please see the source code at https://github.com/iiPythonx/cmd to see
        what data is collected and how it is utilized.

        nd.iipython.dev: this domain serves navidrome, which will log the amount of visitors that
        visit a specific share url. no additional data is collected besides this metric.

        mail.iipython.dev: this is a fully fledged mail server, and thus logs and analyzes all
        incoming mail traffic, logged data includes ip addresses, geographical location, spam list
        status, and more. sending mail to this server automatically assumes acceptance of these
        logging practices.

    3. No data is sold. As stated above, I store nothing personal so there's nothing to sell.

    4. RTBF: If you would like your information deleted from any of these services, please send me
    an email at ben <at> thiswebsite <dot> tld.
